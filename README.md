# vtur-site

Repositorio institucional do `vtur`, destinado ao dominio `vtur.com.br`.

Escopo deste repositorio:

- apresentacao do produto
- explicacao de modulos e fluxo
- planos e precos
- SEO e captacao comercial

O app operacional fica separado em `vtur.app`.

## Stack inicial

- Astro
- HTML/CSS estatico para landing pages
- assets de marca reutilizados em `public/brand`

## Proximos passos

1. Instalar dependencias com `npm install`
2. Subir o ambiente com `npm run dev`
3. Evoluir homepage, paginas de planos, contato e FAQ
